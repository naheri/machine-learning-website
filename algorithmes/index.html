algorithme ?

Selon le CNIL, un algorithme est la description d'une suite d'étapes permettant d'obtenir un résultat à partir d'éléments fournis en entrée. Par exemple, une recette de cuisine est un algorithme permettant d'obtenir un plat à partir de ses ingrédients!

visualisation d'un algorithme "chercheur de chemin"

Pour résumer, les algorithmes de Machine Learning apprennent de manière autonome à effectuer une tâche ou à réaliser des prédictions à partir de données et améliorent leurs performances au fil du temps. Une fois entraîné, l'algorithme pourra retrouver les patterns dans de nouvelles données.

Il existe quatres principales catégories d'algorithmes de Machine Learning. (Apprentissage supervisé, non-supervisé, semi-supervisé et par renforcement). Les algorithmes dits "d'apprentissage supervisé". L'apprentissage supervisé est consiste à apprendre une fonction de prédiction à partir d'exemples annotés.

Apprentissage supervisé

Les algorithmes d'apprentissage supervisé tentent de modéliser les relations et les dépendances entre la sortie de prédiction cible et les caractéristiques d'entrée, de sorte que nous puissions prédire les valeurs de sortie pour les nouvelles données sur la base de ces relations apprises à partir des ensembles de données précédents.

Apprentissage non-supervisé

L'apprentissage non supervisé utilise des algorithmes pour analyser et regrouper des ensembles de données non étiquetées. Ces algorithmes découvrent des modèles cachés ou des regroupements de données sans nécessiter d'intervention humaine. Sa capacité à découvrir les similitudes et les différences entre les informations en fait la solution idéale pour de l'analyse de données ou la reconnaissance d'images.

Apprentissage semi-supervisé

L'apprentissage semi-supervisé implique un petit nombre de données catégorisée et un grand nombre de données non catégorisées. Les problèmes de ce type sont difficiles car ni l'apprentissage supervisé ni l'apprentissage non supervisé n'est capable d'utiliser efficacement les mélanges de données étiquetées et non étiquetées. C'est pourquoi des algorithmes spécialisés d'apprentissage semi-supervisé sont nécessaires.

Apprentissage par renforcement

Avec l'apprentissage par renforcement il s'agit de prendre des mesures pour maximiser la récompense dans une situation particulière. Il est utilisé pour trouver le meilleur comportement possible dans une situation spécifique, il doit donc "apprendre" à partir de son expérience. Parmi les problèmes traités de cette façon, on peut évoquer : apprendre à un robot à marcher, conduire ou accomplir une tâche spécifique (jouer aux échecs), piloter un agent à travers un labyrinthe etc...

L'algorithme AlphaZero a maîtrisé toutes les connaissances liées aux échecs en seulement 4 heures !

Régression

Avant d'aborder les différents algorithmes, il nous clarifier les notions de classification et de regression. La régression est l'approximation d'une fonction (f) des variables(X) à une variable de sortie continue (y). Une variable de sortie continue est une valeur réelle, telle qu'un entier ou d'un flottant. Il s'agit souvent de quantités, telles que des montants et des tailles. Par exemple, on peut prédire qu'une maison sera vendue pour une valeur spécifique.

Classification

La classification est la tâche d'approximation d'une fonction de mappage (f) des variables d'entrée (X) aux variables de sortie discrètes (y). Les variables de sortie sont souvent appelées labels ou catégories. La fonction prédit la classe ou la catégorie pour une observation donnée. Par exemple, un e-mail peut être classé comme appartenant à l'une de deux classes : "spam" et "non spam".

KNN (K Nearest Neighbors)

L'algorithme des k plus proches voisins, également connu sous le nom de KNN ou k-NN, est un discriminant d'apprentissage supervisé non paramétrique, qui utilise la proximité pour effectuer des classifications ou des prédictions sur le regroupement d'un point de données individuel. Utilisé pour des problèmes de régression ou de classification, il est généralement utilisé comme algorithme de classification, en partant de l'hypothèse que des points similaires peuvent être trouvés les uns à côté des autres.

Régression linéaire

L'analyse de régression linéaire est utilisée pour prédire la valeur d'une variable en fonction de la valeur d'une autre variable. La variable que vous voulez prédire est appelée la variable dépendante. La variable que vous utilisez pour prédire la valeur de l'autre variable est appelée la variable indépendante. Elle est utlisée pour prédire une quantité. Ce modèle est simple à comprendre et à utiliser. De ce fait il est utilisé dans beaucoup de domaines différents.

Régression logistique

La régression logistique prédit le résultat d'une variable dépendante catégorique. Par conséquent, le résultat doit être une valeur catégorique ou discrète. Il peut s'agir de Oui ou de Non, de 0 ou de 1, de Vrai ou de Faux, etc. mais au lieu de donner la valeur exacte de 0 et de 1, elle donne les valeurs probabilistes qui se situent entre 0 et 1.

Une machine à vecteurs de support (SVM) est un modèle d'apprentissage automatique supervisé qui utilise des algorithmes de classification pour les problèmes de classification à deux groupes. Après avoir donné à un modèle SVM des ensembles de données d'entraînement étiquetées pour chaque catégorie, il est capable de classer un nouveau texte.
